Base Jump : Image Search Abstraction Layer

User stories:

  1. I can return the image URLs, alt text and page urls for a set of images relating to a given search string and display them.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can return a list of the most recently submitted search strings sorted by timestamp.
